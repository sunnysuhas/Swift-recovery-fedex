# **App Name**: RecoveryAI

## Core Features:

- Case Prioritization: AI-powered prioritization of debt collection cases based on predicted recovery probability and potential value. An LLM 'tool' determines the appropriate case ranking considering debt amount, aging, and historical DCA performance.
- DCA Portal: Dedicated portal for external debt collection agencies to access assigned cases, update statuses, and submit recovery proof.
- Admin Dashboard: Enterprise admin dashboard for real-time visibility into recovery metrics, SLA compliance, aging analysis, and DCA performance comparisons.
- Automated Case Assignment: Intelligent case assignment powered by machine learning recommendations.
- SLA Monitoring: Automated SLA deadline monitoring with breach alerts and automatic escalation rules.
- Audit Logging: Tamper-evident audit logs to capture every user action, ensuring complete traceability for compliance and governance. Saves logs to SQLite
- Reporting and Analytics: Dashboards displaying critical KPIs, including recovery rates, average closure time, aging bucket distribution, and DCA-wise performance rankings with interactive filters and exportable reports.

## Style Guidelines:

- Primary color: Deep blue (#293B93) to convey trust and stability, aligning with the financial context of debt recovery.
- Background color: Light blue (#DDE2F7) for a calm and professional feel.
- Accent color: Teal (#309385) to highlight key actions and important information.
- Body and headline font: 'Inter', a grotesque-style sans-serif with a modern, machined, objective, neutral look; suitable for headlines or body text
- Use clear, professional icons to represent different case statuses, actions, and data points.
- Clean, intuitive layout with clear hierarchy to facilitate efficient navigation and data consumption.
- Subtle animations to provide feedback on user interactions and guide attention to important information.
